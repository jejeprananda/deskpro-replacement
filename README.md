# deskpro-replacement
Pengganti aplikasi Ticketing Despro yang sangat Lambat. CUiiiiih.

bagi kalian yang mau pakai ini silahkan, ini source code opensource jadi silahkan kembangkan

Installasi
1. clone source code ke komputer kalian dengan command
   'git clone https://github.com/jejeprananda/deskpro-replacement.git'
2. diterminal/cmd masuk ke directory source code lalu jalankan 
    'npm i'
3. setelah instalasi selesai tinggal jalankan
    'npm run dev'
4. masuk ke 'localhost:3000' login menggunakan akun agent kalian

oh iya pastikan node versionnya yg terbaru ya (recomended v 22++)

kalau kalian punya NVM
tinggal 'nvm install 22' lalu 'nvm use 22'
